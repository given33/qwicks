# 语音合成 (Text-to-Speech)

当用户希望「朗读文字」「把文字转成语音/音频」「配音」「念出来」或用 `/tts` 命令时，使用本技能。

## 何时使用
- 用户要把一段文字变成可播放的语音。
- 用户希望给内容配音或生成朗读音频。

## 如何调用
调用 `generate_speech` 工具。

## 参数
| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `text` | string | 是 | 要合成语音的文本。 |
| `voice` | string | 否 | 服务商音色 id/名称。不填用设置里的默认音色。 |
| `style` | string | 否 | 音色风格指令（仅部分服务商支持，如小米 MiMo）。 |
| `format` | string | 否 | 音频格式：`mp3`、`wav`、`flac`。 |

## 调用示例
```
generate_speech(text: "欢迎使用语音合成功能，这是生成的朗读音频。", format: "mp3")
```

## 输出处理
- 生成的音频保存到工作区的 `.deepseekgui-audio/` 目录，并作为生成文件返回。
- 在回复里告知用户音频已生成，并可引用文件路径。

## 失败处理
- 若提示「provider 未配置 / missing baseUrl / apiKey / model」：这是内置技能尚未配置。告诉用户前往「设置 → Skills → 内置技能配置」，展开「语音合成」填写 API 凭据后重试；或在 providerId 字段选择已配置凭据的服务商。
- 文本过长导致超限：建议分段合成。
- 网络/超时错误：简要说明并建议重试。
